Reading JavaScript Allongé

April 20, 2019

I am currently reading this amazing book called “JavaScript Allongé” and I would like to use my blog as a notebook to summarize the parts I found the most interesting.

Table of content

As a general rule for effective learning when reading, it is recommended that you read the table of contents and get an idea of what the book contains and how it is structured.

This book’s table of content is a bit messy and some titles are not very descriptive, after separating the wheat from the chaff one could say the content can be split into the following modules:

  • Module 1: The JavaScript functions, their context and wonders.
  • Module 2: Rebinding and references.
  • Module 3: Objects, mutation and state.
  • Module 4: Instances and classes.
  • Module 5: Sequence.
  • Module 6: New ideas.
